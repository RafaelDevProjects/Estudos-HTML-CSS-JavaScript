Python

Python é uma linguagem de programação de alto nível, interpretada, orientada a objetos e amplamente utilizada em desenvolvimento de software, automação, análise de dados, aprendizado de máquina e muitos outros campos. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991.

Variáveis

Variáveis são espaços reservados na memória do computador para que possamos armazenar, temporariamente, um determinado dado. As variaveis devem ter identificadores, podemos dizer que o identificador é o nome que atribuimos a uma variável.

Tipos de Dados

Agora que já foi definido o que são as variáveis, precisamos definir o tipo do dado que será armazenado dentro da variável. A principio, dentro da nossa proposta de programação , podemos, identificar dois grupos básicos de tipos de dados: numéricos e alfanuméricos.

Tipo Numérico

Inteiros (int): Os números inteiros em Python são representados pelo tipo int. Eles podem ser positivos, negativos ou zero. Por exemplo: 5, -10, 0.

Flutuantes (float): Os números de ponto flutuante são representados pelo tipo float. Eles incluem números decimais e números com notação científica. Exemplo: 3.14, 2.0e-3.

Complexos (complex): Números complexos são representados pelo tipo complex. Eles consistem em uma parte real e uma parte imaginária, ambas sendo números de ponto flutuante. Exemplo: 2 + 3j.

Tipos Alfanuméricos

Strings (str): As strings são sequências de caracteres alfanuméricos e são representadas pelo tipo str. Elas podem ser definidas usando aspas simples (' '), aspas duplas (" "), ou até mesmo aspas triplas para strings multilinha. Exemplo: "Olá, mundo!", 'Python'.

Listas (list): As listas são coleções ordenadas e mutáveis de objetos, incluindo números, strings e outros tipos. Elas são representadas pelo tipo list e são definidas usando colchetes [ ]. Exemplo: [1, 2, 3], ['a', 'b', 'c'].

Tuplas (tuple): Tuplas são semelhantes às listas, mas são imutáveis, ou seja, seus elementos não podem ser modificados após a criação. Elas são definidas usando parênteses ( ). Exemplo: (1, 2, 3), ('apple', 'banana', 'cherry').

Dicionários (dict): Os dicionários são estruturas de dados que associam chaves (keys) a valores (values) e são representados pelo tipo dict. Cada chave é única. Eles são definidos usando chaves { }. Exemplo: {'nome': 'João', 'idade': 30}.

Conjuntos (set): Conjuntos são coleções não ordenadas de elementos únicos e são representados pelo tipo set. Eles são definidos usando chaves { } ou a função set(). Exemplo: {1, 2, 3}, set([1, 2, 2, 3, 3]).

Booleanos (bool): Os tipos booleanos representam os valores verdadeiro (True) e falso (False). Eles são frequentemente usados para expressar condições lógicas. Exemplo: True, False.

Pedindo um valor ao usuário

Usamos o comando input para pedir um valor ao usúario, lembre-se de colocar o tipo de valor antes do input.

numero = int(input('Digite um número: '))

Tomada de Decisões

Tomada de decisões em Python, como em qualquer linguagem de programação, envolve a capacidade de criar caminhos diferentes para o código, dependendo das condições específicas. Isso é feito usando estruturas de controle de fluxo, principalmente as declarações if, elif (opcional), e else

Declaração if: A declaração if é usada para testar uma condição. Se a condição for avaliada como verdadeira, o bloco de código indentado sob o if é executado. Caso contrário, o bloco é ignorado.

Declaração elif (opcional): A declaração elif (abreviação de "else if") é usada para testar condições adicionais após um if. Ela permite avaliar múltiplas condições sequencialmente até encontrar uma que seja verdadeira.

Declaração else (opcional): A declaração else é usada para definir um bloco de código que será executado caso nenhuma das condições anteriores seja verdadeira.

Operadores Lógicos: Python oferece operadores lógicos como and (e), or (ou), e not (não) para combinar e manipular condições em declarações if.

Aninhamento de Decisões: É possível aninhar várias declarações if, elif e else para criar estruturas de decisão mais complexas. No entanto, deve-se ter cuidado para manter a clareza do código.

nota = 85
if nota >= 70:
    if nota >= 90:
        print("Aprovado com distinção.")
    else:
        print("Aprovado.")
else:
    print("Reprovado.")

Laçoes de Repetições

Em Python, os laços de repetição, também conhecidos como estruturas de controle de fluxo de repetição, são usados para executar um bloco de código repetidamente enquanto uma condição é verdadeira ou para iterar sobre uma sequência de elementos. Existem duas principais construções para isso: o for e o while.

    Laço while

  • O laço while é usado para repetir um bloco de código enquanto uma condição especificada for verdadeira.
  • É importante garantir que a condição do while seja eventualmente falsa para evitar loops infinitos.
    • contador = 0
while contador < 5:
    print(contador)
    contador += 1

Neste exemplo, o while executa o bloco de código enquanto a variável contador for menor que 5.

    Laço For

  • O laço for é usado para iterar sobre uma sequência (como uma lista, tupla, string ou range).
  • Ele executa um bloco de código para cada item na sequência.
for i in range(5):
    print(i)

Neste exemplo, o for itera sobre os números de 0 a 4 e imprime cada valor.

Listas

Em Python, uma lista é uma estrutura de dados que permite armazenar uma coleção ordenada de elementos. As listas são muito versáteis e amplamente utilizadas na programação. Para criar uma lista, você pode usar colchetes [ ] e inserir os elementos separados por vírgulas.

Acesso a Elementos: Você pode acessar os elementos de uma lista usando índices (começando em 0 para o primeiro elemento). Por exemplo:

primeiro_elemento = minha_lista[0]  # Retorna 1

Para saber o tamanho (número de elementos) de uma lista, use a função len(). Por exemplo:

tamanho = len(minha_lista)  # Retorna 5

Você pode adicionar elementos a uma lista usando o método append() para adicionar um elemento ao final da lista ou o método insert() para inserir um elemento em uma posição específica.

minha_lista.append(6)      # Adiciona 6 ao final
minha_lista.insert(2, 7)   # Insere 7 na posição 2

Para verificar se um elemento está em uma lista, use o operador 'in'.

if 3 in minha_lista:
    print("3 está na lista")

Para remover elementos, você pode usar o método remove() para remover um elemento específico ou a instrução del para remover um elemento com base no índice.

minha_lista.remove(4)  # Remove o elemento 4
del minha_lista[1]     # Remove o elemento na posição 1

Listas Aninhadas: Você pode criar listas dentro de listas, formando listas aninhadas (ou listas multidimensionais).

matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

Funções

Funções em Python são blocos de código reutilizáveis que executam uma tarefa específica. Elas ajudam a organizar e modularizar seu código, tornando-o mais legível e fácil de manter.

Definição da função

Você pode definir uma função usando a palavra-chave def, seguida pelo nome da função e parênteses que podem conter argumentos.

def saudacao(nome):
    print(f"Olá, {nome}!")

Chamando Funções

Para executar uma função, você a chama pelo nome e passa os argumentos necessários entre parênteses.

saudacao("Alice")

Argumentos

Funções podem ter zero ou mais argumentos. Os argumentos são valores que você passa para a função para que ela os processe.

def soma(a, b):
    return a + b

Retorno de valores

Uma função pode retornar um valor usando a palavra-chave return. Isso permite que você obtenha resultados da função para uso posterior.

resultado = soma(3, 5)

Funções Recursivas

Funções podem chamar a si mesmas. Isso é útil para resolver problemas que podem ser divididos em subproblemas semelhantes.

def fatorial(n):
    if n == 0:
        return 1
    else:
        return n * fatorial(n-1)

Dicionários

Dicionários em Python são estruturas de dados que permitem armazenar e organizar pares de chave-valor. Aqui está um resumo das principais características dos dicionários em Python:

Definição de Dicionário

Um dicionário é definido usando chaves {} e contém pares de chave-valor separados por dois-pontos :.

meu_dicionario = {"chave1": "valor1", "chave2": "valor2"}

Acesso a Valores

Você pode acessar os valores de um dicionário fornecendo a chave correspondente entre colchetes [].

print(meu_dicionario["chave1"])  # Saída: "valor1"

Modificação de Valores

Os valores de um dicionário podem ser alterados atribuindo um novo valor a uma chave existente.

meu_dicionario["chave2"] = "novo_valor"

Adição de Novos Pares

Você pode adicionar novos pares chave-valor ao dicionário simplesmente atribuindo um valor a uma chave inexistente.

meu_dicionario["chave3"] = "valor3"

Remoção de Pares

Use a palavra-chave del para remover um par chave-valor específico.

del meu_dicionario["chave1"]

Verificação de Existência

Você pode verificar se uma chave existe em um dicionário usando o operador in.

if "chave2" in meu_dicionario:
    print("A chave2 existe no dicionário.")

Iteração em Dicionários

É possível iterar pelos pares chave-valor de um dicionário usando loops for.

for chave, valor in meu_dicionario.items():
    print(f"A chave {chave} tem o valor {valor}.")

Métodos Úteis

Dicionários em Python têm métodos úteis, como keys(), values(), e items(), que retornam as chaves, valores e pares chave-valor, respectivamente.

chaves = meu_dicionario.keys()
    valores = meu_dicionario.values()
    pares = meu_dicionario.items()

Tuplas

Em Python, uma tupla é um tipo de estrutura de dados que permite armazenar um conjunto ordenado de elementos.

  1. Imutabilidade: As tuplas são imutáveis, o que significa que, uma vez criadas, não é possível alterar, adicionar ou remover elementos individuais.
  2. Sintaxe: As tuplas são definidas usando parênteses, por exemplo: (1, 2, 3). Também é possível criar uma tupla sem parênteses, desde que haja uma vírgula para separar os elementos: 1, 2, 3.
  3. Elementos Ordenados: Os elementos em uma tupla são ordenados e podem ser acessados por meio de índices, começando em 0. Por exemplo, para acessar o segundo elemento de uma tupla t, você usaria t[1].
  4. Heterogeneidade: As tuplas podem conter elementos de diferentes tipos de dados, como inteiros, strings, float, etc.
  5. Comprimento Fixo: Uma vez criada, o tamanho de uma tupla é fixo e não pode ser alterado.
  6. Packing e Unpacking: Você pode criar uma tupla empacotando vários valores em uma única variável e, em seguida, desempacotá-la para obter os valores individuais. Por exemplo: 
    tupla = (1, 2, 3)
a, b, c = tupla  # Isso desempacota os valores para a, b e c
  7. Iteração: Você pode percorrer os elementos de uma tupla usando um loop for ou acessá-los diretamente por índice.
  8. Métodos Básicos: As tuplas suportam métodos básicos, como count para contar ocorrências de um elemento e index para encontrar o índice de um elemento.

Exemplo de uso de tuplas:

# Criando uma tupla
frutas = ('maçã', 'banana', 'laranja')
# Acessando elementos print(frutas[0]) # Saída: 'maçã' # Iterando pela tupla for fruta in frutas: print(fruta) # Empacotando e desempacotando tupla = 1, 2, 3 # Empacotando a, b, c = tupla # Desempacotando # Verificando o comprimento comprimento = len(frutas) # Retorna 3 # Métodos count_maçã = frutas.count('maçã') # Retorna 1 index_banana = frutas.index('banana') # Retorna 1

As tuplas são frequentemente usadas quando você deseja armazenar um conjunto de valores relacionados que não devem ser alterados durante a execução do programa, como coordenadas x e y, informações sobre data e hora, etc.

Manipulando Arquivos

A manipulação de arquivos em Python é uma tarefa comum em programação, e o Python oferece diversas funcionalidades para lidar com arquivos de diferentes tipos, como leitura, escrita, criação e exclusão de arquivos. Aqui estão os conceitos básicos de como funciona a manipulação de arquivos em Python:

Abertura de Arquivos:

Você pode abrir um arquivo usando a função open( ). Esta função aceita dois argumentos: o nome do arquivo que você deseja abrir e o modo de abertura (leitura, escrita, etc.).

arquivo = open("meu_arquivo.txt", "r")

Modos de abertura comuns:

  • "r": leitura (padrão).
  • "w": escrita, cria um novo arquivo ou sobrescreve o arquivo existente.
  • "a": anexação, adiciona dados ao final de um arquivo existente.
  • "b": modo binário para arquivos (por exemplo, "rb" ou "wb").
  • "x": cria o arquivo para escrita, mas gera um erro se o arquivo já existir.

Leitura de Arquivos:

Para ler o conteúdo de um arquivo, você pode usar métodos como read(), readline(), ou readlines(). Exemplo de leitura de um arquivo linha por linha:

with open("meu_arquivo.txt", "r") as arquivo:
    for linha in arquivo:
        print(linha)

Escrita em Arquivos:

Para escrever em um arquivo, use o modo "w" ou "a" ao abrir o arquivo e, em seguida, use o método write() para adicionar conteúdo ao arquivo.

with open("meu_arquivo.txt", "w") as arquivo:
    arquivo.write("Este é um exemplo de escrita em arquivo.\n")

Fechamento de Arquivos:

Sempre feche o arquivo após terminar de usá-lo para liberar os recursos do sistema. Você pode fazer isso explicitamente usando o método close() ou, preferencialmente, usando um bloco with (context manager), que fecha automaticamente o arquivo quando você sai do bloco.

with open("meu_arquivo.txt", "r") as arquivo:
    conteudo = arquivo.read()
# O arquivo é automaticamente fechado ao sair do bloco "with".

Manipulação de Arquivos Binários:

Para arquivos binários, como imagens, use o modo "rb" para leitura e "wb" para escrita.

with open("imagem_origem.jpg", "rb") as origem, open("imagem_destino.jpg", "wb") as destino:
    destino.write(origem.read())

Manipulação de Diretórios:

O módulo os e o módulo os.path fornecem funcionalidades para trabalhar com diretórios, como criar, renomear e excluir diretórios.

import os
os.mkdir("novo_diretorio")

Lembre-se de que é importante lidar com exceções ao trabalhar com arquivos, pois podem ocorrer erros durante a leitura ou escrita. Portanto, sempre use estruturas de controle, como try e except, para tratar possíveis exceções relacionadas à manipulação de arquivos.

Manipulação de Strings

Manipular strings em Python é uma tarefa comum e importante, uma vez que as strings são usadas para representar texto. Python oferece uma ampla gama de métodos e operadores para realizar várias operações em strings. Abaixo, você encontrará algumas das operações de manipulação de strings mais comuns em Python:

Concatenação de Strings:

Para combinar duas ou mais strings, você pode usar o operador + ou o método str.join().

str1 = "Olá"
str2 = "Mundo"
resultado = str1 + " " + str2  # Concatenação com o operador +
resultado = " ".join([str1, str2])  # Concatenação com o método join()

Comprimento de uma String:

Para obter o número de caracteres em uma string, você pode usar a função len().

texto = "Python é uma linguagem de programação"
comprimento = len(texto)  # Retorna 31

Acessar Caracteres em uma String:

Você pode acessar caracteres em uma string usando índices.

texto = "Python"
primeiro_caracter = texto[0]  # Retorna 'P'

Fatiamento de Strings:

Para obter partes específicas de uma string, você pode usar a notação de fatiamento.

texto = "Python é incrível"
parte = texto[7:10]  # Retorna "é "

Métodos de Busca e Substituição:

Você pode usar métodos como str.find(), str.replace(), str.count(), str.startswith(), e str.endswith() para buscar e substituir substrings, contar ocorrências e verificar se uma string começa ou termina com um determinado prefixo ou sufixo.

texto = "Python é uma linguagem de programação"
texto.find("linguagem")  # Retorna a posição onde "linguagem" começa (12)
novo_texto = texto.replace("Python", "Java")  # Substitui "Python" por "Java"

Divisão de strings

Você pode dividir uma string em substrings usando o método str.split().

frase = "Python é uma linguagem de programação"
palavras = frase.split()  # Divide a string em palavras

Metodos de Transformação

Existem métodos de string para transformar o caso, como lower(), upper(), e title()

Remover Espaços em Branco:

Você pode remover espaços em branco no início e no final de uma string usando str.strip() ou remover espaços em branco à esquerda ou à direita usando str.lstrip() ou str.rstrip().

texto = "   Python   "
texto_stripped = texto.strip()  # Retorna "Python"

Verificação de Tipo de Caracteres:

Os métodos isalpha(), isdigit(), isalnum(), entre outros, verificam o tipo de caracteres em uma string.

Manipulando arquivos JSON

Arquivos JSON (JavaScript Object Notation) são uma forma popular de armazenar e transmitir dados estruturados em formato de texto. Em Python, você pode manipular arquivos JSON usando a biblioteca integrada chamada json. Aqui está um resumo de como trabalhar com arquivos JSON em Python:

Importe a biblioteca JSON:

import json

Abrir um arquivo JSON para leitura:

Para abrir um arquivo JSON existente para leitura, você pode usar a função open e, em seguida, o método json.load para carregar o conteúdo do arquivo em um objeto Python.

with open('arquivo.json', 'r') as arquivo:
    dados = json.load(arquivo)

Manipular dados JSON:

Após carregar os dados do arquivo JSON em uma estrutura de dados Python (normalmente um dicionário), você pode manipulá-los como faria com qualquer outra estrutura de dados Python.

Abrir um arquivo JSON para escrita:

Para criar ou sobrescrever um arquivo JSON, você pode usar a função open com o modo de escrita 'w' e, em seguida, o método json.dump para escrever um objeto Python no arquivo JSON.

dados = {"nome": "João", "idade": 30}
    with open('arquivo.json', 'w') as arquivo:
        json.dump(dados, arquivo)

Formatação de saída:

Você pode especificar a formatação do arquivo JSON para torná-lo mais legível, definindo o argumento indent no json.dump. Isso adicionará espaçamento e recuo ao arquivo JSON.

with open('arquivo_formatado.json', 'w') as arquivo:
    json.dump(dados, arquivo, indent=4)

Tratamento de erros:

Certifique-se de lidar com exceções ao trabalhar com arquivos JSON, como FileNotFoundError ao tentar abrir um arquivo inexistente ou json.JSONDecodeError ao tentar carregar dados JSON inválidos.

Convesando com o sistema operacional

A biblioteca platform em Python fornece informações gerais sobre o sistema operacional e o ambiente de execução. Ela é útil para obter informações básicas sobre o sistema em que o código está sendo executado. Aqui estão alguns dos principais comandos disponíveis na biblioteca platform:

  1. platform.system(): Retorna o nome do sistema operacional. Isso pode ser "Linux", "Windows", "Darwin" (para macOS), "Java", entre outros. 
    import platform

sistema = platform.system()
print("Sistema Operacional:", sistema)
  2. platform.release(): Retorna a versão do sistema operacional. 
    import platform

versao = platform.release()
print("Versão do Sistema Operacional:", versao)
  3. platform.machine(): Retorna a arquitetura da máquina, como "i386", "x86_64", "arm64", etc. 
    import platform

arquitetura = platform.machine()
print("Arquitetura da Máquina:", arquitetura)
  4. platform.version(): Retorna informações detalhadas sobre a versão do sistema operacional, geralmente incluindo informações adicionais sobre o ambiente de sistema. 
    import platform

informacoes_versao = platform.version()
print("Informações de Versão:", informacoes_versao)
  5. platform.architecture(): Retorna a arquitetura do sistema, como "32 bits" ou "64 bits" 
    import platform

arquitetura = platform.architecture()
print("Arquitetura do Sistema:", arquitetura)
  6. platform.node(): Retorna o nome da máquina de rede (nome do host) no qual o código Python está sendo executado. 
    import platform

nome_do_host = platform.node()
print("Nome do Host:", nome_do_host)

Protocolos de Comunicação e Python

Em Python, você pode implementar protocolos de comunicação usando várias abordagens, dependendo dos requisitos e das tecnologias envolvidas. Alguns dos protocolos de comunicação mais comuns em Python incluem:

TCP/IP (Transmission Control Protocol/Internet Protocol)

O TCP/IP é um protocolo amplamente utilizado para comunicação na Internet e em redes locais. Você pode usar a biblioteca padrão socket do Python para criar aplicativos que se comunicam via TCP/IP. O socket permite que você crie sockets do lado do cliente e do lado do servidor para enviar e receber dados através da rede.

import socket

# Lado do servidor
servidor = "127.0.0.1"
porta = 43210
server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server_socket.bind((servidor, porta))
server_socket.listen(1)
client_socket, client_endereco = server_socket.accept()
dado_recebido = client_socket.recv(1024)
client_socket.send(b"Ola, cliente!")
client_socket.close()

# Lado do cliente
servidor = "127.0.0.1"
porta = 43210
msg = bytes(input('Digite algo: '), 'utf-8')
client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
client_socket.connect((servidor, porta))
client_socket.send(msg)
responsta = client_socket.recv(1024)
client_socket.close()

UDP (User Datagram Protocol)

Em resumo, o UDP é uma escolha adequada para aplicações que requerem baixa latência e não são sensíveis à perda ocasional de pacotes. No entanto, é importante notar que o UDP não é adequado para todas as situações, especialmente para aplicações que exigem confiabilidade na entrega de dados. Nesses casos, o TCP, com seu controle de congestionamento e garantia de entrega, é uma escolha mais apropriada.

import socket
#lado do servidor
# Cria um socket UDP
server_socket = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

# Associa o socket a um endereço IP e porta
server_address = ("localhost", 12345)
server_socket.bind(server_address)

print(f"Servidor aguardando em {server_address}")

while True:
    data, client_address = server_socket.recvfrom(1024)
    print(f"Recebido {len(data)} bytes de {client_address}: {data.decode()}")

    # Responde ao cliente
    response = "Mensagem recebida no servidor!"
    server_socket.sendto(response.encode(), client_address)

Neste exemplo, o servidor cria um socket UDP e o associa a um endereço IP e porta. Em seguida, ele entra em um loop onde aguarda pacotes UDP. Quando um pacote é recebido, ele imprime os dados e o endereço do cliente e envia uma resposta de volta para o cliente.

import socket
#Lado do Cliente
# Cria um socket UDP
client_socket = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

# Define o endereço do servidor
server_address = ("localhost", 12345)

# Dados a serem enviados
message = "Olá, servidor!"
client_socket.sendto(message.encode(), server_address)

# Recebe a resposta do servidor
data, server_address = client_socket.recvfrom(1024)
print(f"Recebido {len(data)} bytes de {server_address}: {data.decode()}")

# Fecha o socket do cliente
client_socket.close()

O cliente cria um socket UDP e envia uma mensagem para o servidor especificado por seu endereço IP e porta. Em seguida, ele espera pela resposta do servidor e imprime a resposta recebida.

FTP (File Transfer Protocol)

O File Transfer Protocol (FTP) é um protocolo utilizado para a transferência de arquivos entre sistemas em uma rede, comumente a Internet. Ele permite que os usuários façam upload e download de arquivos de e para servidores FTP, que são dispositivos ou servidores específicos configurados para armazenar e compartilhar arquivos.

from ftplib import FTP

# Conectar ao servidor FTP
ftp = FTP("ftp.example.com")
ftp.login(user="seu_usuario", passwd="sua_senha")

# Diretório de destino no servidor FTP
diretorio_destino = "/pasta_no_servidor"

# Nome do arquivo a ser enviado
nome_arquivo = "arquivo.txt"

# Abra o arquivo em modo binário e faça o upload
with open(nome_arquivo, "rb") as arquivo:
    ftp.storbinary(f"STOR {diretorio_destino}/{nome_arquivo}", arquivo)

# Encerrar a conexão FTP
ftp.quit()

Programação orientada a objetos (OOP)

A programação orientada a objetos (POO) em Python é uma abordagem de programação que organiza o código em objetos que possuem atributos (dados) e métodos (funções) associados. Aqui está um resumo das principais características e conceitos da POO em Python:

  1. Classes e objetos
    2.
    • Uma classe é um modelo para criar objetos. Ela define os atributos e métodos que os objetos desse tipo terão
    • Um objeto é uma instância de uma classe. Pode ser criado usando a sintaxe nome_da_classe().
    class Pessoa:
def __init__(self, nome, idade):
    self.nome = nome
    self.idade = idade

# Criando um objeto Pessoa
p1 = Pessoa("João", 30)
print(p1.nome)  # Saída: João
print(p1.idade)  # Saída: 30
  2. Atributos
    3.
    • São variáveis que pertencem a um objeto e armazenam dados sobre o objeto.
    • Podem ser acessados usando a notação de ponto (objeto.atributo)
    class Carro:
    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo

# Atributos de objetos Carro
carro1 = Carro("Toyota", "Corolla")
print(carro1.marca)  # Saída: Toyota
print(carro1.modelo)  # Saída: Corolla
  3. Metodos:
    4.
    • São funções definidas dentro de uma classe que podem ser chamadas para realizar operações em objetos daquela classe.
    • Podem ser acessados usando a notação de ponto (objeto.atributo).
    class Retangulo:
    def __init__(self, comprimento, largura):
        self.comprimento = comprimento
        self.largura = largura

    def calcular_area(self):
        return self.comprimento * self.largura

# Método de um objeto Retangulo
r1 = Retangulo(5, 4)
print(r1.calcular_area())  # Saída: 20
  4. Metodos Especiais(Mágicos):
    5.
    • São métodos predefinidos em Python que permitem que você execute operações específicas em objetos, como inicialização, representação, adição, subtração, etc.
    • Eles têm nomes que começam e terminam com duplo sublinhado (por exemplo, __init__, __str__, __add__)
    lass Livro:
    def __init__(self, titulo, autor):
        self.titulo = titulo
        self.autor = autor

    def __str__(self):
        return f"{self.titulo} por {self.autor}"

# Método especial __str__
livro = Livro("Python for Dummies", "John Smith")
print(livro)  # Saída: Python for Dummies por John Smith
