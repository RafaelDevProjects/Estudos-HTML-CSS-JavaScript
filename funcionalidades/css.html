CSS

Estilo inline

Os estilos inline são feitos dentro das linhas do html, não é muito ulilizada por causa da bagunça que fica as linhas de codigo

Estilos internos

No metodo de estilos locais, colocar a tag <style> dentro do <head> e colocar a classe desejada
como a classe body ou a classe h1, isso fara com que todos as tags <h1> mudem

Desvantagens

  • Não da para mudar em multiplos documentos
  • O codigo das css fica extenso

Esse tipo de metodo é recomendado quando se tem apenas um documento

Estilos externos

O estilo mais versatil, serve para multiplos documentos

Criar um arquivo style.css e adiconar um estilo nesse novo arquivo. Na aba do <head> no seu domento html, adicionar a tag <link> de css e fazer o link com o arquivo de estilos css

Exportando fontes externas

Baixando fontes externas

Uma fonte externa pode ser baixada em um site como o dafonte.com

Baixar a fonte desejada, e adicionar no seu codigo o comando @font-face

exemplo:

@font-face{
    font-family: 'Coffe';  
    scr: url('fontes/CoffeCake.otf') format ('opentype'),
    url('fontes/CoffeCake.ttf') format ('truetype');
    font-weight:normal;
    font-style:normal;
}

Há varios tipos de formatos de arquivo de fontes, os mais usados são otf e ttf

Colocar a fonte no seu css usando o nome que você deu para ela, no exemplo o nome da fonte é 'Coffe'

Seletores personalizados

Usando id no css

Muitas vezes o programador se depara com seletores em seu html que são iguais, como varios h1 ou outros seletores, mas e se ele quiser mudar somente um h1 em expecífico?

Para isso é usado o id dentro do html

Usar a tag id para personalizar um seletor com o nome desejado, usar o nome escolhido do seletor para muda-lo no css

No css o comando id quer dizer #, portanto, usar no css o comando #nomeescolhido

Usando class no css

O comando class e o comando id, fazem basicamente a mesma coisa, porem, o comando id pode ser usado em apenas um elemento, o comando class pode ser usado em multiplos elementos. No css ele é identificado com um '.nomeescolhido'

Usar o comando <span class= "nome"> para criar uma class no meio de escritas

    Modelo de caixas

    As caixas ficam envolta do elemento como um elemento h1, existem dois modelos de caixa, as caixas box-level e as inline-level

    As caixas podem ter:

    • height
    • weight
    • border
    • outline
    • padding
    • margin

    box-level

    Essas caixas pulam a linha.

    box-level

    A parte azul escuro é a borda e entre o conteudo e a borda é o padding, a parte de fora da borda é aonde fica a margem.

    Inline-level

    Essas caixas ficam na propria linha inline-level desse jeito.

    A parte azul escuro é a borda e entre o conteudo e a borda é o padding, a parte de fora da borda é aonde fica a margem.

    Mudando o tamanho da imagem

    As imagens que são vinculadas ao background-image, já vem com um tamanho, para mudar o tamanho de uma imagem usar o comando background-image,se for necessario preencher a tela inteira da sua view port do body, usar o comando background-image: cover pode se a melhor forma, porém a imagem não sera 100% mostrada na tela, partes dala ficarão de fora.

    Para mostrar 100% da imagem é necessario usar o comando background-image:content, isso faz com que o todo o conteudo da imagem seja mostrado, porém, pode aparecer bordas no seu site.

    exemplo de fundo cover

    Com essas configurações surge outro problema, a imagem estara andando junto do conteúdo e quando tiver muito conteúdo isso criara um novo problema, por isso é usando o comando background-attachment: fixed, desse jeito o plano de fundo ficara fixado.

    Usando tabelas para dados

    As tabelas são ultilizadas para dados e podem ser criadas no HTML:

    A1 B1 C1
    A2 B2 C2
    A3 B3 C3

    Essa é uma tabela simples, foi feita usando as tags:

    • <table>
    • <tr>
    • <td>
    • <th>

    Usando dessa forma:

    <table>
    <tr>
        <td>A1</td>
        <td>B1</td>
        <td>C1</td>
    </tr>
</table>

    nesse exemplo esta apenas a primeira linha da tabela

    Também foi usado o css para criar as bordas, como comando border, e a junção das bordas como comando border-collapse, também foi usado o width para arrumar a lagura da tabela, dentro das <td>

    Uso de escopo

    O escopo não é algo que muda visualmente a sua tabela, porém, os mecanismos de buca dos navegadores vão reconhecer mais facil oque é cada coluna e linha no seu site

    O scope:"col" é usado em colunas do seu th, como no exemplo:

    <th scope="col">estado</th>

    O scope:"row" é usado para linhas do th, como no exemplo:

    <td scope="row">Total de habitantes</td>

    Efeito zebrado

    O efeito zabrado ajuda na leitura de uma tabela, pode ser feito no css com a pseudo classe :nth-child(), usando a variavel odd e a even.

    tbody > tr:nth-child(odd){
    background-color: lightgray;
}

    Mesclagem de células

    As mesclagens de células são muito importantes para posicionar os dados da tabela, fazendo com que um dado possa ocupar duas colunas ou mais ou até duas linhas ou mais.

    A C
    D E F
    G H 
    <td colspan="2">A</td>
    <td rowspan="2">F</td>

    Usando o comando colspan="2" a letra 'A' ficou o cupando duas colunas da tabela.

    Usando o comando rowspan="2" a letra 'F' ocupou duas linhas da tabela.

    Media Queries

    As Media Queries são usadas para mostrar seu site em diferentes lugares de mídia, fazendo com que o conteúdo seja consumivel no seu computador ou no seu celular.

    Para usar as Media Queries é necessário criar um link de css dentro da head, e adionar o tipo de midia desejada, como para screen que são para telas ou para print, que são para impressoras.

    Impressoras

    Para criar um estilo especifico para impressoras é preciso analisar o site e ver que estilos sao necessarios para a leirura do seu site em papel.

    Como o comando display:none que tira algo do site, como o header, aperte ctrl + P para ver como o site fica no modo de impressão.

    O pseudo elemento ::after pode ser bastante útil também, usando para colocar um conteudo ápos o artigo com o comando content adicionando entre aspas um conteudo, como a url do site por exemplo.

    Portrait

    Também podemos usar as Media quearies para diferenciar quando o aparelho esta no modo retrato. Usando o comando portrait na área de midia nos links, como no exemplo a baixo.

    <link rel="stylesheet" href="estilo/print.css" media="screen and (orientation: portrait)">

    Landscape

    Assim como o portrait, podemos usar a orientação landscape, quando a tela estiver no modo paisagem.

    <link rel="stylesheet" href="estilo/print.css" media="screen and (orientation: landscape)">

    Typical Device Breakpoints

    Eles são pontos de quebra de pixel de uma midia para outra, como por exemplo, as telas dos celulares possuem uma quantidade de pixel diferente dos monitores.

      os breakpoins são:

    • tela pequena: até 600px
    • celular: 600px até 768px
    • tablete: 768px até 992px
    • desktop: 992px até 1200px
    • tela grande: acima de 1200px
