Linguaguem C#

Bem-vindo ao mundo da programação com C#! Como uma linguagem de programação moderna e versátil, o C# é amplamente reconhecido pela sua elegância e poder. Desenvolvido pela Microsoft, o C# é parte integrante do ecossistema .NET, oferecendo uma sintaxe limpa e uma ampla gama de recursos para criar aplicativos robustos e escaláveis. Se você está começando sua jornada na programação ou já possui experiência em outras linguagens, o C# oferece uma curva de aprendizado suave e uma vasta comunidade de apoio. Prepare-se para mergulhar em conceitos fundamentais como tipos de dados, estruturas de controle e orientação a objetos, enquanto descobre o potencial desta linguagem para criar desde simples aplicativos de desktop até soluções empresariais complexas.

Console.WhiteLine

Em C#, a tarefa de imprimir ou exibir informações na tela é realizada principalmente através do uso do método Console.WriteLine(). Este método aceita uma variedade de tipos de dados como argumentos e exibe o valor correspondente no console. Por exemplo, para exibir uma simples mensagem de texto, você pode usar: 

Console.WriteLine("Olá, mundo!");

Se você quiser exibir o valor de uma variável, basta passar a variável como argumento para Console.WriteLine(). Por exemplo: 

int numero = 10;
Console.WriteLine("O valor de numero é: " + numero);

Aplicando a Orientação a objetos

Para criar uma classe em c# usamos a palavra class, que deve ter um nome com a primeira letra em maiúsculo para mostrar que é uma class, podemos criar atributos para cada característica dessa classe, como o exemplo a seguir: 

class Musica        
{
    public string nome;
    public string artista;
    public string duracao;
    public bool disponivel;
}

Para criar uma variavel que seja da classe Musica precisamos do seguinte codigo: 

Musica musica1 = new Musica();

Para acessar um desses atributos na lista precisamos que ela seja publica, para acessar: 

musica1.nome = "Convoque seu buda";

Desse modo o nome da Musica1 tera o nome Convoque seu buda

Podemos também criar comportamentos dentro das classes que são os metodos, como a mostrada seguir: 

public void ExibirFichaTecnica()
{
    Console.WriteLine($"Nome: {nome}");
    Console.WriteLine($"Artista: {artista}");
    Console.WriteLine($"Duração: {duracao}");
}

musica1.ExibirFichaTecnica();
