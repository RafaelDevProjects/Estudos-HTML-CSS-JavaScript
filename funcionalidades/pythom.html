Pythom

Python é uma linguagem de programação de alto nível, interpretada, orientada a objetos e amplamente utilizada em desenvolvimento de software, automação, análise de dados, aprendizado de máquina e muitos outros campos. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991.

Variáveis

Variáveis são espaços reservados na memória do computador para que possamos armazenar, temporariamente, um determinado dado. As variaveis devem ter identificadores, podemos dizer que o identificador é o nome que atribuimos a uma variável.

Tipos de Dados

Agora que já foi definido o que são as variáveis, precisamos definir o tipo do dado que será armazenado dentro da variável. A principio, dentro da nossa proposta de programação , podemos, identificar dois grupos básicos de tipos de dados: numéricos e alfanuméricos.

Tipo Numérico

Inteiros (int): Os números inteiros em Python são representados pelo tipo int. Eles podem ser positivos, negativos ou zero. Por exemplo: 5, -10, 0.

Flutuantes (float): Os números de ponto flutuante são representados pelo tipo float. Eles incluem números decimais e números com notação científica. Exemplo: 3.14, 2.0e-3.

Complexos (complex): Números complexos são representados pelo tipo complex. Eles consistem em uma parte real e uma parte imaginária, ambas sendo números de ponto flutuante. Exemplo: 2 + 3j.

Tipos Alfanuméricos

Strings (str): As strings são sequências de caracteres alfanuméricos e são representadas pelo tipo str. Elas podem ser definidas usando aspas simples (' '), aspas duplas (" "), ou até mesmo aspas triplas para strings multilinha. Exemplo: "Olá, mundo!", 'Python'.

Listas (list): As listas são coleções ordenadas e mutáveis de objetos, incluindo números, strings e outros tipos. Elas são representadas pelo tipo list e são definidas usando colchetes [ ]. Exemplo: [1, 2, 3], ['a', 'b', 'c'].

Tuplas (tuple): Tuplas são semelhantes às listas, mas são imutáveis, ou seja, seus elementos não podem ser modificados após a criação. Elas são definidas usando parênteses ( ). Exemplo: (1, 2, 3), ('apple', 'banana', 'cherry').

Dicionários (dict): Os dicionários são estruturas de dados que associam chaves (keys) a valores (values) e são representados pelo tipo dict. Cada chave é única. Eles são definidos usando chaves { }. Exemplo: {'nome': 'João', 'idade': 30}.

Conjuntos (set): Conjuntos são coleções não ordenadas de elementos únicos e são representados pelo tipo set. Eles são definidos usando chaves { } ou a função set(). Exemplo: {1, 2, 3}, set([1, 2, 2, 3, 3]).

Booleanos (bool): Os tipos booleanos representam os valores verdadeiro (True) e falso (False). Eles são frequentemente usados para expressar condições lógicas. Exemplo: True, False.

Pedindo um valor ao usuário

Usamos o comando input para pedir um valor ao usúario, lembre-se de colocar o tipo de valor antes do input.

numero = int(input('Digite um número: '))

Tomada de Decisões

Tomada de decisões em Python, como em qualquer linguagem de programação, envolve a capacidade de criar caminhos diferentes para o código, dependendo das condições específicas. Isso é feito usando estruturas de controle de fluxo, principalmente as declarações if, elif (opcional), e else

Declaração if: A declaração if é usada para testar uma condição. Se a condição for avaliada como verdadeira, o bloco de código indentado sob o if é executado. Caso contrário, o bloco é ignorado.

Declaração elif (opcional): A declaração elif (abreviação de "else if") é usada para testar condições adicionais após um if. Ela permite avaliar múltiplas condições sequencialmente até encontrar uma que seja verdadeira.

Declaração else (opcional): A declaração else é usada para definir um bloco de código que será executado caso nenhuma das condições anteriores seja verdadeira.

Operadores Lógicos: Python oferece operadores lógicos como and (e), or (ou), e not (não) para combinar e manipular condições em declarações if.

Aninhamento de Decisões: É possível aninhar várias declarações if, elif e else para criar estruturas de decisão mais complexas. No entanto, deve-se ter cuidado para manter a clareza do código.

nota = 85
if nota >= 70:
    if nota >= 90:
        print("Aprovado com distinção.")
    else:
        print("Aprovado.")
else:
    print("Reprovado.")

Laçoes de Repetições

Em Python, os laços de repetição, também conhecidos como estruturas de controle de fluxo de repetição, são usados para executar um bloco de código repetidamente enquanto uma condição é verdadeira ou para iterar sobre uma sequência de elementos. Existem duas principais construções para isso: o for e o while.

    Laço while

  • O laço while é usado para repetir um bloco de código enquanto uma condição especificada for verdadeira.
  • É importante garantir que a condição do while seja eventualmente falsa para evitar loops infinitos.
    • contador = 0
while contador < 5:
    print(contador)
    contador += 1

Neste exemplo, o while executa o bloco de código enquanto a variável contador for menor que 5.

    Laço For

  • O laço for é usado para iterar sobre uma sequência (como uma lista, tupla, string ou range).
  • Ele executa um bloco de código para cada item na sequência.
    • for i in range(5):
    print(i)

Neste exemplo, o for itera sobre os números de 0 a 4 e imprime cada valor.
