Linguaguem C++

Imprimindo mensagem na tela

Para imprimir uma mensagem na tela ultilizamos a biblioteca std para usar a funcionalidade cout (saída) e endl (fim da linha). Como mostra o exemplo a seguir:

#include <iostream>
using namespace std;
    
    int main () {
        cout << "*************************************" << endl;
        cout << "* Bem-vindos ao C++ *" << endl;
        cout << "*************************************" << endl;
    }

Mensagem com variavel

Para imprimir uma mensagem que tenha uma variável dentro dela é simples, precisamos apenas alterar um pouco o codigo anterior, como mostra o codigo a seguir:


int numero_secreto = 42;
cout << "O numero secreto e" << numero_secreto <<". Nao conte para ninguem" << endl;
